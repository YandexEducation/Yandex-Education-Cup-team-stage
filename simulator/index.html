Перед запуском симулятора сделайте три шага:

  1. Слева в блоке «Загрузка скрипта» выберите файл .py со своим кодом.
  2. Слева в блоке «Загрузка карты» выберите файл карты .json или откройте «🗺 Редактор карты», нарисуйте трассу и нажмите «💾 Сохранить».
  3. Когда выбран и скрипт, и карта, кнопка «🎮 Запустить симулятор» станет активной — нажмите её.

В симуляторе для начала движения нажмите кнопку «Старт».

Симулятор сам запускает скрипт. Нажмите «Старт» для начала.

Подсказки
  • Функция входа: Используйте def run_robot(bot):
  • Задержки: Вызов bot.sleep(seconds) — не забыли про yield
  • Совместимость: Код работает одинаково в браузере и на реальном роботе
  • Карта из редактора подхватывается симулятором после «Сохранить»
  • «Сохранить в файл» — скачать карту в JSON; «Загрузить карту» — загрузить другую версию карты
  • Редактор карты: Черные полосы — линия для датчиков. Ластик — зажать и вести. Зоны GO/STOP/LEFT/RIGHT — детекция камеры. Старт — клик задаёт позицию (0° вправо, 90° вверх).
