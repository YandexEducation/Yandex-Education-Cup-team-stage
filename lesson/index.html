1. Сенсоры и исполнительные устройства робота

Ниже — принципы работы каждого компонента, вызовы в программе и их имитация в симуляторе.

Датчики линии (IR)

Инфракрасный LED излучает свет вниз, фотодиод принимает отражённый от поверхности свет. Белая поверхность даёт высокое значение (до 4095), чёрная линия — низкое (близко к 0). Три датчика (левый, центр, правый) позволяют роботу понимать, где он относительно линии. На показания могут влиять яркий внешний свет, высота и угол датчика.

В программе: bot.line_left.read(), bot.line_sensor.read(), bot.line_right.read(). В симуляторе: анализ цвета пикселей карты под позицией робота.

Инфракрасный датчик расстояния

Датчик излучает ИК-луч вперёд и измеряет отражённый свет. Близкий объект — высокое значение; дальний — низкое (до ~80 см). На показания также влияют внешний свет, цвет и фактура объекта (тёмные/матовые отражают хуже), угол наклона. Датчик имеет небольшую погрешность, поэтому его лучше использовать для определения близких препятствий.

В программе: bot.sharp.read() или bot.sharp.distance_cm(). В симуляторе: трассировка луча по карте до границы полигона. Результат в сантиметрах или в виде сырого значения.

Энкодеры

На оси колеса — диск с прорезями; ИК-излучатель и приёмник. При вращении приёмник выдаёт импульсы. Два канала (A, B) определяют направление. Подсчёт импульсов даёт пройденное расстояние. Проскальзывание колёс и грязь на диске могут искажать показания.

В программе: bot.left_encoder.read(), bot.right_encoder.read(). В симуляторе: счётчики обновляются по скорости колёс; при сбросе обнуляются.

Камера (распознавание знаков)

Матрица фиксирует свет; изображение 96×96 в оттенках серого подаётся в нейросеть, которая распознаёт STOP, GO, LEFT, RIGHT. На результат влияют освещение, блики, тени, угол и расстояние до знака, а так же работа светодиодов. В симуляторе освещение не учитывается.

В программе: bot.camera.detect_sign(). В симуляторе: зоны (круги) на карте с метками; при попадании робота в зону возвращается соответствующий знак.

Моторы

Электродвигатель постоянного тока; драйвер (H-мост) управляет направлением и скоростью через PWM. move(left, right) задаёт скорость колёс от -100 до 100. На реальную скорость влияют заряд батареи, нагрузка, разброс характеристик моторов.

В программе: bot.motors.move(left, right), bot.motors.stop(). В симуляторе: модель робота перемещается по карте согласно заданным скоростям.

LED-лента

RGB-светодиоды отображают заданный цвет.

В программе: bot.leds.fill((R, G, B)), bot.leds.write(). В симуляторе: четыре точки на модели робота отображают цвет ленты.

Вид спереди: камера (4), ИК-датчики (2), кнопка (7), Led подсветка (6)

Вид снизу: три датчика линии (1), моторы с энкодерами (3,5)

2. Практическая часть

В этом разделе: примеры кода по каждому компоненту робота и практические задания.

Чему научитесь: управлять моторами и паузами (yield), читать датчики линии и расстояния, обрабатывать знаки камеры, менять цвет LED, писать скрипт следования по линии и реакции на знаки.

2.1. Работа с компонентами робота

Скрипт робота — функция def run_robot(bot):. Симулятор и робот вызывают её при старте. Параметр bot — объект с доступом ко всем устройствам (моторы, датчики, камера, LED). Функция должна быть генератором: для паузы пиши yield bot.sleep(секунды) — иначе симулятор не обновит позицию и датчики.

Моторы

bot.motors.move(left, right) — скорость от -100 до 100. Оба плюс — вперёд, оба минус — назад. Разные знаки — поворот. bot.motors.stop() — остановка.

Пример:

def run_robot(bot):
    bot.motors.move(50, 50)      # вперёд
    yield bot.sleep(1.0)         # обязательно yield!
    bot.motors.move(-30, 30)     # поворот влево
    yield bot.sleep(0.5)
    bot.motors.stop()

Датчики линии

read() возвращает 0–4095. Белая ≈ 4095, чёрная линия ≈ 0. Порог ~3500: ниже — датчик над линией. Три датчика: bot.line_left, bot.line_sensor, bot.line_right.

Пример:

line_left = bot.line_left.read()
line_center = bot.line_sensor.read()
line_right = bot.line_right.read()
on_line = line_center < 3500  # центр видит чёрную линию

Датчик расстояния

bot.sharp.distance_cm() — расстояние в см (до ~80). bot.sharp.read() — сырое значение 0–4095 (близко = высокое).

Пример:

dist = bot.sharp.distance_cm()
if dist < 15:
    bot.motors.stop()  # препятствие близко

Камера (знаки)

bot.camera.detect_sign() возвращает "STOP", "GO", "LEFT", "RIGHT" или None.

Пример:

sign = bot.camera.detect_sign()
if sign == "STOP":
    bot.motors.stop()
    yield bot.sleep(2.0)
elif sign == "LEFT":
    bot.motors.move(40, -40)
    yield bot.sleep(0.5)

LED-лента

bot.leds.fill((R, G, B)) — цвет 0–255 на канал. bot.leds.write() — применить.

Пример:

RED = (255, 0, 0)
GREEN = (0, 255, 0)
bot.leds.fill(RED)
bot.leds.write()

Детекция STOP

Камера распознаёт знаки: bot.camera.detect_sign() возвращает "STOP", "GO", "LEFT", "RIGHT" или None. В примере: робот едет вперёд, при виде знака STOP — останавливается на 2 секунды.

Пример:

def run_robot(bot):
    while True:
        sign = bot.camera.detect_sign()
        if sign == "STOP":
            bot.motors.stop()
            yield bot.sleep(2.0)
        else:
            bot.motors.move(40, 40)
        yield bot.sleep(0.05)

Задача 2. Следование по линии

Напишите скрипт движения между черных линий.

Напишите скрипт: робот едет по белой области между черных линий. Читай три датчика. Если левый на линии — поворот вправо; правый — влево; три датчика на линии — стоп; иначе — вперёд. В цикле yield bot.sleep(0.05).

Порог 3500. Используй elif on_left, elif on_right.
def run_robot(bot):
    while True:
        line_left = bot.line_left.read()
        line_center = bot.line_sensor.read()
        line_right = bot.line_right.read()
        on_left = line_left < 3500
        on_right = line_right < 3500
        if on_left and on_right:
            bot.motors.stop()
            yield bot.sleep(0.3)
        elif on_left:
            bot.motors.move(-10, 10)
        elif on_right:
            bot.motors.move(10, -10)
        else:
            bot.motors.move(30, 30)
        yield bot.sleep(0.05)

Задача 3. Реакция на знаки

Добавьте к задаче 2: проверяй bot.camera.detect_sign(). STOP — остановка 2 сек; GO — вперёд 1.5 сек; LEFT — поворот влево 0.5 сек; RIGHT — вправо 0.5 сек. Используй continue после обработки знака.

В начале цикла: if sign == "STOP": — остановить, yield bot.sleep(2), continue.
def run_robot(bot):
    while True:
        line_left = bot.line_left.read()
        line_right = bot.line_right.read()
        sign = bot.camera.detect_sign()
        on_left, on_right = line_left < 3500, line_right < 3500
        if sign == "STOP":
            bot.motors.stop()
            yield bot.sleep(2.0)
            continue
        elif sign == "GO":
            bot.motors.move(50, 50)
            yield bot.sleep(1.5)
            continue
        elif sign == "LEFT":
            bot.motors.move(40, -40)
            yield bot.sleep(0.5)
            continue
        elif sign == "RIGHT":
            bot.motors.move(-40, 40)
            yield bot.sleep(0.5)
            continue
        if on_left and on_right:
            bot.motors.stop()
            yield bot.sleep(0.3)
        elif on_left:
            bot.motors.move(-10, 10)
        elif on_right:
            bot.motors.move(10, -10)
        else:
            bot.motors.move(30, 30)
        yield bot.sleep(0.05)

Задача 4. LED-подсветка

Добавьте к задаче 3: при старте мигание зелёный → красный → жёлтый → синий (по 1 сек). В цикле: на STOP — красный, на GO — зелёный, на LEFT/RIGHT — жёлтый, иначе — выключить.

bot.leds.fill((R, G, B)), bot.leds.write(). Цвета: RED=(255,0,0), GREEN=(0,255,0), YELLOW=(255,255,0).
def run_robot(bot):
    RED, GREEN = (255,0,0), (0,255,0)
    YELLOW, BLUE, OFF = (255,255,0), (0,0,255), (0,0,0)
    for color in [GREEN, RED, YELLOW, BLUE]:
        bot.leds.fill(color)
        bot.leds.write()
        yield bot.sleep(1.0)
        bot.leds.fill(OFF)
